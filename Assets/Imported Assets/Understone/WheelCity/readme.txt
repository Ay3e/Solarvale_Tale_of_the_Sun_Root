USAGE

To use the Rolling Town model, you can simply drag the model (in folder "Models") or the prefab (in folder "Prefabs") into the scene.

Additionally, the prefab has a simple animation (also available in folder "Animations"). If you do not require it, simply remove the component from the prefab.
